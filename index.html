Network Switch Configuration and Security

Project Overview

In the realm of networking, configuring switches and ensuring robust security measures are crucial for maintaining network stability and safeguarding against potential threats. This project focuses on essential configurations including VLAN trunking, spanning tree protocol optimization, and security implementations across a network of switches.

Objective

This project aims to configure network switches (Core1, D-SW1, D-SW2, A-SW1, A-SW2) to enable VLAN trunking, optimize spanning tree protocol settings, and implement critical security measures.

Equipment Required

  • Network switches and associated cables
  • Terminal emulation software (e.g., PuTTY)
  • Basic understanding of networking principles and configuration

Configuration Steps

Initial Setup and Basic Configuration

Switch> enable
Switch# configure terminal
Switch(config)# hostname Core1
Core1(config)# exit

VLAN Trunking Configuration

Core1(config)# interface range gi0/1
Core1(config-if)# switchport mode dynamic desirable
Core1(config-if)# exit
Core1(config)# interface range gi0/2
Core1(config-if)# switchport mode dynamic desirable
Core1(config-if)# exit
Core1(config)# interface range gi0/1 - 2
Core1(config-if-range)# switchport mode trunk

Spanning Tree Protocol (STP) Optimization

Core1(config)# spanning-tree vlan 1 root primary

STP Attack Prevention

A-SW1(config)# interface range fa0/1 - 24
A-SW1(config-if-range)# spanning-tree portfast
A-SW1(config-if-range)# spanning-tree bpduguard enable
A-SW1(config-if-range)# exit
A-SW2(config)# interface range fa0/1 - 24
A-SW2(config-if-range)# spanning-tree portfast
A-SW2(config-if-range)# spanning-tree bpduguard enable
A-SW2(config-if-range)# exit

Broadcast Storm Control

A-SW1(config)# interface gi0/2
A-SW1(config-if)# storm-control broadcast level 50
A-SW1(config-if)# exit
A-SW2(config)# interface gi0/2
A-SW2(config-if)# storm-control broadcast level 50
A-SW2(config-if)# exit

Port Security Implementation

A-SW1(config)# interface fa0/1
A-SW1(config-if)# switchport mode access
A-SW1(config-if)# switchport port-security
A-SW1(config-if)# switchport port-security maximum 1
A-SW1(config-if)# switchport port-security violation shutdown
A-SW1(config-if)# switchport port-security mac-address sticky
